# SKILL: REPOSICIONAMENTO ESTRATÉGICO DE SUB-MERCADO (V2.0)

## VISÃO GERAL

Esta skill identifica em qual sub-mercado o avatar já está gastando dinheiro (mesmo frustrado) e cria a ponte de migração para a "Nova Oportunidade" — o produto como solução estruturalmente diferente das anteriores. O objetivo não é competir dentro de uma categoria existente, mas criar uma nova categoria em que somos líderes por definição.

**NOVIDADE V2.0:** Adiciona o Sistema de Migração Identitária, o Protocolo de Assassinato do Método Atual (10 argumentos estruturados), o Mapa de Congregações para segmentação de audiência e o Framework de Contranarrativa por nicho.

---

## GATILHO DE ATIVAÇÃO

Use quando o usuário pedir:
- "Como competir com [concorrente dominante]"
- "Meu avatar já usa [método concorrente] — como convencê-lo"
- "Como criar um ângulo diferente no nicho saturado de [X]"
- "Como fazer o avatar abandonar o que já usa"
- "Reposicionamento do meu produto"

O usuário fornecerá: `[Produto/Método]` + `[Sub-mercado atual do avatar]` + `[Nicho]`.

---

## PROTOCOLO DE EXECUÇÃO

---

### FASE 1: MAPEAMENTO DAS TENTATIVAS ATUAIS

**1.1 — Os 10+ Métodos/Produtos que o Avatar Já Usa ou Comprou:**

Para cada método, identificar:
- Nome do método/produto
- Quanto o avatar gasta (estimativa mensal/anual)
- Nível de satisfação (1-10)
- A promessa que fez o avatar comprar
- A frustração principal que ficou

**Tabela de Mapeamento:**

| Método | Gasto estimado | Satisfação | Promessa original | Frustração principal |
|--------|----------------|------------|-------------------|----------------------|
| [Método A] | R$[X]/mês | X/10 | [Prometia Y] | [Frustração Z] |

**1.2 — Ordenar por "Dor Não Resolvida × Gasto":**
O sub-mercado com maior gasto + menor satisfação = maior oportunidade de reposicionamento.

**1.3 — Identificação das Congregações:**
Onde o avatar busca informação, se reúne e compartilha frustações sobre esses métodos:
- Fóruns específicos do nicho (Reddit, grupos de Facebook, Telegram)
- Canais do YouTube e os comentários recorrentes
- Comentários nas redes sociais de concorrentes
- Grupos de WhatsApp de practitioners
- Eventos e comunidades do sub-mercado

*Estas congregações são onde os anúncios de reposicionamento devem aparecer.*

---

### FASE 2: SELEÇÃO DO SUB-MERCADO ALVO

Critérios de seleção (pontuar 1-5 cada):

| Critério | O que avaliar |
|----------|---------------|
| **Dor Não Resolvida** | O gasto é alto mas a frustração continua? |
| **Linguagem Específica** | Existem gírias negativas sobre o método? ("isso não funciona mais", "é tudo enrolação") |
| **Vergonha de Continuar** | O avatar sente vergonha de ainda usar/precisar desse método? |
| **Abertura para Alternativa** | Existem sinais de busca por saída? (pesquisas, perguntas em grupos) |
| **Tamanho da Audiência** | Quantas pessoas estão neste sub-mercado? |

**Sub-mercado selecionado:** O de maior pontuação total.

**Declaração de posicionamento:**
```
"[Avatar] está gastando R$[X]/mês em [sub-mercado] que promete [resultado]
mas entrega [frustração]. Nossa oportunidade é a migração desse dinheiro
para [nossa solução] que resolve o que [sub-mercado] nunca conseguiu."
```

---

### FASE 3: PROTOCOLO DE ASSASSINATO DO MÉTODO ATUAL (10 Argumentos)

**Objetivo:** Criar 10 argumentos persuasivos e específicos que provam por que o método atual é estruturalmente incapaz de entregar o resultado prometido.

**Tipos de argumentos (usar mix dos 5):**

**Tipo 1 — Ineficiência Estrutural:**
O método ataca o sintoma, não a causa. Mesmo seguindo perfeitamente, o problema volta.
```
"[Método X] promete [resultado]. Mas [resultado] é um sintoma de [causa raiz].
Tratar o sintoma sem tratar a causa é como [analogia].
É por isso que [o resultado não se mantém / o problema volta]."
```

**Tipo 2 — Obsolescência:**
O método foi criado para um contexto que não existe mais.
```
"[Método X] foi desenvolvido quando [contexto histórico].
Hoje [contexto mudou]. O que funcionava em [época] produz [resultado frustrante] em [hoje]."
```

**Tipo 3 — Efeito Colateral Oculto:**
O método resolve um problema mas cria outro que ninguém fala.
```
"O que ninguém conta sobre [Método X]: para funcionar, ele exige [efeito colateral].
Para [avatar], isso significa [consequência negativa específica] — que é exatamente
o que ele estava tentando evitar."
```

**Tipo 4 — Premissa Falsa:**
O método é baseado em uma lógica que nunca foi verdadeira para este avatar.
```
"[Método X] parte da premissa de que [premissa].
O problema: [para avatar específico], [por que a premissa não se aplica].
Resultado: [consequência frustrante inevitável]."
```

**Tipo 5 — Evidência de Mercado:**
Os próprios usuários do método provam que não funciona.
```
"[Número] de avaliações de [Método X] no [Reclame Aqui/Trustpilot/YouTube]
revelam o mesmo padrão: [padrão de reclamação].
Não é coincidência. É estrutural."
```

**Gerar 2 argumentos de cada tipo = 10 argumentos totais.**

---

### FASE 4: CRIAÇÃO DA NOVA NARRATIVA (A Nova Oportunidade)

**4.1 — O Posicionamento de Radical:**
Posicionar como quem traz a verdade que o sub-mercado esconde.

*Não somos "melhor que X" — somos "a razão pela qual X nunca podia funcionar revelada".*

**4.2 — A Estrutura da Nova Oportunidade:**

```
NARRATIVA DE REPOSICIONAMENTO:

"Você tentou [método 1], [método 2], [método 3].
Não funcionou — e você ficou se perguntando por quê.

A resposta: todos esses métodos compartilham [falha estrutural comum].
Eles prometem [resultado] mas constroem sobre [premissa incorreta].

[Nossa solução] parte de uma lógica completamente diferente:
em vez de [o que todos fazem], [o que fazemos] porque [razão lógica].

Isso não é 'mais do mesmo com outro nome.'
É o motivo pelo qual [resultado que os outros não entregam] finalmente se torna possível."
```

**4.3 — A Migração Identitária:**
O reposicionamento não é só de produto — é de identidade. O avatar precisa se ver como alguém que "evoluiu além de" o sub-mercado antigo.

| Identidade Antiga | Identidade Nova |
|-------------------|----------------|
| "Alguém que usa [método X]" | "Alguém que entende por que [método X] falha" |
| "Alguém que ainda não conseguiu" | "Alguém que estava usando a abordagem errada" |
| "[Perfil de estagnação]" | "[Perfil de quem descobriu a real causa]" |

**Frase de transição identitária:**
```
"Deixe de ser [identidade antiga] para se tornar [identidade nova].
Não porque você mudou — mas porque finalmente sabe o que estava errado."
```

---

### FASE 5: ESTRATÉGIA DE DISTRIBUIÇÃO DO REPOSICIONAMENTO

**5.1 — Copy para as Congregações:**
Para cada congregação identificada na Fase 1, criar:
- Hook específico para o contexto (não o mesmo gancho em todos os lugares)
- Tom calibrado para o nível de frustração do grupo
- CTA que leva para o conteúdo de reposicionamento (não direto para venda)

**5.2 — Sequência de Conteúdo de Migração (5 peças):**

| Peça | Objetivo | Formato | Mensagem central |
|------|----------|---------|-----------------|
| 1 | Validar a frustração | Post/Vídeo curto | "Não é culpa sua que [método X] não funcionou" |
| 2 | Nomear a falha estrutural | Artigo/Vídeo médio | "O motivo real pelo qual [método X] nunca podia funcionar" |
| 3 | Apresentar nova lógica | Vídeo/Live | "A abordagem que resolve o que [método X] ignora" |
| 4 | Prova social de migrantes | Depoimentos | "Como [perfil] parou de usar [método X] e conseguiu [resultado]" |
| 5 | A Nova Oportunidade | VSL/LP | Oferta completa com narrativa de reposicionamento |

**5.3 — Ângulo de Anúncio para Audiência do Concorrente:**
Para quem está comprando/consumindo o concorrente agora:
```
"Se você usa [método concorrente] e [frustração específica]...
existe uma razão para isso. E ela não tem nada a ver com você."
```

## PROTOCOLO DE HANDOFF (MANDATÓRIO)

1. **Sub-mercado Alvo:** Com justificativa baseada nos scores.
2. **Argumento de Reposicionamento #1:** O mais forte dos 10.
3. **Frase de Transição Identitária:** Pronta para usar em copy.
4. **Sugestão:** `"Ative a SKILL-DEVASTADOR V4.0 usando esta Narrativa de Reposicionamento no Bloco 2 (Validação do Mártir + Counter-Positioning)"`.
5. **Transferência de Contexto:** Narrativa de reposicionamento + 10 argumentos + Congregações identificadas.
