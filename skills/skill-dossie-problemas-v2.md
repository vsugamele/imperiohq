# SKILL: DOSSIÊ DE PROBLEMAS DE ALTO IMPACTO (V2.0)

## VISÃO GERAL

Esta skill é especializada em identificar, mapear e priorizar os problemas reais e urgentes de um avatar — separando "incômodos toleráveis" de "dores que impedem a vida de continuar". O output é um arsenal estratégico para copy, anúncios e posicionamento de oferta.

**NOVIDADE V2.0:** Sistema de pontuação expandido para 7 critérios (score máximo 35), mapeamento completo em 13 categorias de impedimentos, Gatilhos de Voyerismo com template expandido e integração direta com o Avatar Architect V7.0.

---

## INPUTS NECESSÁRIOS

Confirmar antes de executar:
1. Avatar (nome, perfil resumido, situação atual)
2. Nicho e produto/serviço
3. Dossiê do Avatar V7.0 (se disponível — para cruzar com dados)

Se o Dossiê do Avatar não estiver disponível, executar esta skill com base nas informações fornecidas e sinalizar lacunas.

## METODOLOGIA DE EXECUÇÃO

---

### ETAPA 1: MAPEAMENTO COMPLETO DE IMPEDIMENTOS (13 Categorias)

Para cada categoria, listar mínimo 8-10 itens específicos ao nicho e avatar. Nunca usar linguagem genérica — cada problema deve ser reconhecível e doloroso para o avatar ao ler.

**1. PROBLEMAS PRINCIPAIS**
Os obstáculos centrais que bloqueiam o resultado desejado.

**2. SUB-PROBLEMAS**
Derivações e manifestações técnicas dos problemas principais.

**3. DORES EMOCIONAIS**
As feridas geradas pelos problemas físicos/financeiros. Usar linguagem visceral — como o avatar se sente, não o que acontece com ele.

**4. BARREIRAS**
Obstáculos externos que impedem a resolução (acesso, custo, tempo, recursos).

**5. MEDOS**
Projeções negativas que paralisam a ação. Separar em:
- Medos de fazer (tentar e piorar)
- Medos de não fazer (consequências da inação)
- Medos de exposição (ser visto falhando)

**6. DESCULPAS**
Justificativas internas que o avatar usa para adiar. São os autoenganos identificados no Framework de Trauma do Avatar Architect.

**7. OBJEÇÕES**
Resistências verbalizadas à solução (o que ele diz quando alguém tenta vender).

**8. DISTRAÇÕES**
O que o avatar faz ao invés de resolver o problema real (onde o foco vai errado).

**9. LIMITAÇÕES**
Capacidades reais ausentes — técnicas, cognitivas, financeiras ou relacionais.

**10. FALTAS**
Recursos, conhecimentos, ferramentas ou suporte que não possui.

**11. IMPEDIMENTOS**
Bloqueios estruturais externos além do controle imediato do avatar.

**12. DESGASTE EMOCIONAL**
O custo psicológico acumulado de carregar o problema por meses ou anos.

**13. PRESSÕES SOCIAIS**
Expectativas externas que amplificam a dor interna (família, pares, clientes, mercado).

---

### ETAPA 2: SISTEMA DE PONTUAÇÃO 7 CRITÉRIOS (Score Máximo: 35)

Pontuar cada problema de **1 a 5** em cada critério:

| Critério | O que mede | Score |
|----------|------------|-------|
| **Nível de Dor** | Força emocional e/ou física do problema | 1-5 |
| **Desejo de Resolver** | O quanto o avatar quer ativamente a solução | 1-5 |
| **Piora sem Ação** | Impacto negativo comprovável de não resolver | 1-5 |
| **Velocidade de Resolução** | Quão rápido o produto/método pode resolver | 1-5 |
| **Disposição de Pagar** | O avatar já gasta (ou gastaria) para resolver? | 1-5 |
| **Facilidade de Comunicar** | O quanto o problema é verbalizável em copy | 1-5 |
| **Frequência de Ocorrência** | Com que regularidade o avatar sente/enfrenta isso | 1-5 |

**Regras de pontuação:**
- Score 5 em Dor: o avatar pensa nisso diariamente, tem sintoma físico associado
- Score 5 em Disposição de Pagar: já tentou resolver com dinheiro antes
- Score 5 em Facilidade de Comunicar: consegue ser dito em uma frase que o avatar reconhece instantaneamente
- Score 1 em Piora: o problema se resolve sozinho com o tempo
- Se não houver evidência, pontuar 3 e sinalizar com flag: *[sem evidência direta]*

**Ordenar do maior para o menor score total.**

**OS 15 CAMPEÕES:** Selecionar os 15 com maior pontuação e destacar:
- Por que cada um é campeão (justificativa de 2-3 frases)
- Qual critério foi o diferencial decisivo
- Qual categoria de copy ele alimenta melhor (headline, abertura, objeção, urgência)

---

### ETAPA 4: GATILHOS DE VOYERISMO — MOMENTOS DE IDENTIFICAÇÃO INSTANTÂNEA

Selecionar **8 situações de alto impacto emocional** vividas no dia a dia do avatar para criar conexão imediata no primeiro segundo de leitura/visualização.

Para cada situação, usar o template completo:

```
[NOME DO MOMENTO] — Intensidade: X/10
- Situação: [descrição sensorial específica — onde, quando, com quem]
- Sintoma Físico: [reação corporal exata — "frio na barriga", "nó na garganta", "calor no rosto"]
- Pensamento Automático: "[a frase exata que passa pela cabeça, entre aspas]"
- Comportamento Resultante: [o que o avatar faz imediatamente depois]
- Categoria de Copy: [onde usar — headline / abertura / objeção / urgência / fechamento]
```

**Exemplos de qualidade (usar como referência de especificidade):**
- ❌ Fraco: "A insegurança de não saber o que fazer"
- ✅ Forte: "O frio na barriga quando uma cliente cacheada agenda horário e você já começa a rezar para dar certo"

- ❌ Fraco: "A frustração de cobrar pouco"
- ✅ Forte: "O momento de hesitar antes de falar o preço para um cliente novo, e na última hora cobrar R$20 a menos porque travou"

---

### ETAPA 5: MATRIZ DE PROBLEMAS POR ÂNGULO DE COPY

Organizar os 15 campeões por melhor uso estratégico:

**HEADLINES DE CHOQUE (Top 3 de maior Dor + Comunicabilidade):**
Para os três problemas com maior score combinado nestes dois critérios, escrever 1 headline de teste no formato da Skill Devastador Cat 1.

**ABERTURAS EMPÁTICAS (Top 3 de maior Frequência + Dor):**
Os problemas que ocorrem com mais regularidade e dor são os melhores para aberturas — o avatar os sente agora, hoje, neste momento.

**OBJEÇÕES ANTECIPADAS (Top 3 de maior score em Desculpas + Objeções):**
Os problemas mapeados nas categorias Desculpas e Objeções com maior pontuação revelam as resistências que o copy precisa quebrar preventivamente.

**ARGUMENTOS DE URGÊNCIA (Top 3 de maior Piora):**
Os problemas com maior score em "Piora sem Ação" são a base da urgência legítima — mostram o que acontece se o avatar continuar parado.

**GARANTIA E REVERSÃO DE RISCO (Top 3 de maior Medo):**
Os medos com maior pontuação revelam o que a garantia precisa cobrir explicitamente para transferir o risco.

---

### ETAPA 6: INSIGHTS ESTRATÉGICOS FINAIS

**6.1 — O Problema Raiz (The Root Cause):**
Qual dos 15 campeões é a CAUSA dos outros? O problema que, se resolvido, dissolveria os demais automaticamente?

**6.2 — O Problema Mais Vendável (The Hook Problem):**
Qual tem a combinação perfeita de Dor alta + Comunicabilidade alta + Frequência alta? Este é o anzol para anúncios de topo de funil.

**6.3 — O Problema de Maior ROI Emocional:**
Qual tem o maior gap entre "custo de não resolver" e "custo da solução"? Este é o argumento central para justificar o preço.

**6.4 — O Problema Invisível (The Hidden Pain):**
Qual problema o avatar não percebe conscientemente mas sente todos os dias? Nomear esse problema gera a conexão mais visceral.

**6.5 — Palavras Exatas do Avatar:**
Com base nas categorias Desculpas, Objeções e Dores Emocionais, listar 10-15 frases exatas que o avatar usaria ao descrever o problema. Estas frases devem aparecer literalmente no copy — não parafraseadas.
